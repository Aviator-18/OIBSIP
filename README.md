## 🔧 Running Projects

### For Console Applications
```bash
javac -d bin src/*.java
java -cp bin MainClass
```

### For JavaFX Applications
```bash
javac --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml -d bin src/*.java
java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml -cp bin:lib/* MainClass
```

### For Database Projects
1. Set up MySQL database
2. Run SQL scripts from `database/` folder
3. Configure database credentials in config files
4. Run the application

## 🐛 Troubleshooting

### Common Issues and Solutions

#### Issue: "JavaFX runtime components are missing"
**Solution:** Add JavaFX to module path with VM options:
```
--module-path "path/to/javafx/lib" --add-modules javafx.controls,javafx.fxml
```

#### Issue: "MySQL connection failed"
**Solution:** 
- Verify MySQL service is running
- Check credentials in DatabaseConfig.java
- Ensure port 3306 is not blocked

#### Issue: "ClassNotFoundException: com.mysql.cj.jdbc.Driver"
**Solution:** Add MySQL Connector JAR to project libraries

For more help, check individual project READMEs or create an [issue](https://github.com/Aviator-18/OIBSIP/issues).
